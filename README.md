### Quick Deployment
1. **Fork/Clone** this repository
2. **Upload Files** to your GitHub repository:
   - `AMET_Campus_WiFi_Proposal.html`
   - `ARARAT Logo (3) (3).png`
   - `README.md`
3. **Enable GitHub Pages**:
   - Go to repository Settings
   - Scroll to "Pages" section
   - Select "Deploy from a branch"

   - Click "Save"
